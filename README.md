## Project Overview
Analyzed movie dataset to answer business questions:
- Most frequent genres
- Highest popularity by genre
- Vote distribution trends
- Release year patterns

Dataset: 9,827 movies (mymoviedb.csv)

## Key Insights
- Drama: 3,715 entries (top genre)
- Spider-Man No Way Home: 5,083 popularity peak
- Balanced vote categories (~2,400 each)
- Memory: 691KB to 749KB optimized
